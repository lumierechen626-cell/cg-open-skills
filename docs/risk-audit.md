# 脱敏风险审计

## 高风险项

| 风险 | 处理 |
|---|---|
| 本地路径 | 全部删除，用“用户自定义目录”替代。 |
| 私有 reference | 不进入公开仓库。 |
| 真实客户 / 学员案例 | 全部改成虚构案例或结构占位。 |
| 固定品牌标签 | 改成 `{你的品牌}`。 |
| 自动发布 / 服务器配置 | 不进入公开版。 |
| 商业成交 SOP | 只保留通用流程，不保留内部细节。 |

## 当前版本处理结果

- `wechat-article-chain-lite` 已删除原版本地路径、内部文档引用、自动推送配置。
- `insurance-private-domain-moments` 已把固定品牌标签改为可配置占位符。
- `business-insight-public` 已删除内部商业文档路径。
- 所有 Skill 均不包含 API Key、Token、Secret 和服务器地址。
