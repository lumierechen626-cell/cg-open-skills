# 下载方案

## 主入口：GitHub

GitHub 负责：

- 版本管理
- Release 下载
- Issue 反馈
- Star / Fork 传播

推荐每个正式版本发一个 Release，并上传：

- `cg-open-skills-vX.Y.Z.zip`
- `manifest.json`
- 更新日志

## 国内入口：静态镜像

国内镜像只做下载页，不做复杂系统。

页面至少包含：

- 最新版本下载
- 单个 Skill 下载
- 更新日志
- 安装说明
- GitHub 主仓库链接
- 飞书文档兜底链接

## 兜底入口：飞书文档

飞书文档适合不会 GitHub 的用户：

- 放下载链接
- 放安装截图
- 放常见问题
- 放商业合作入口

飞书不作为主仓库，不承担版本管理。
