# 安全与脱敏规则

公开前必须检查：

- 不含本地绝对路径，例如 `/Users/...`。
- 不含 API Key、Token、Secret、App ID。
- 不含服务器 IP、数据库地址、内部系统地址。
- 不含真实客户、真实学员、真实成交记录。
- 不含私有素材库和内部 reference。
- 不含不能公开传播的版权材料。

## reference 分级

| 类型 | 是否公开 | 处理方式 |
|---|---|---|
| 通用方法论 | 可以 | 保留 |
| 虚构示例 | 可以 | 保留 |
| 脱敏案例 | 谨慎 | 确认无法反推个人 |
| 私有口风表 | 不公开 | 删除或替换成通用规则 |
| 真实客户资料 | 不公开 | 删除 |
| 内部 SOP | 不公开 | 只保留通用流程 |
