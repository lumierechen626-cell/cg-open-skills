# 安装说明

## 方式 1：复制单个 Skill

1. 打开 `skills/` 下对应目录。
2. 复制该目录到你的 Agent skills 目录。
3. 确认目录内至少有 `SKILL.md`。
4. 重启或刷新你的 Agent。

示例：

```text
skills/private-domain/insurance-private-domain-moments/
```

复制到：

```text
你的 Agent skills 目录/insurance-private-domain-moments/
```

## 方式 2：下载 zip

发布页会提供：

- `cg-open-skills-v0.1.1.zip`：全部第一批 Skill。
- `single-skill-*.zip`：单个 Skill。

解压后按方式 1 复制。

## 使用前必须替换

这些占位符需要改成你自己的信息：

- `{你的品牌}`
- `{你的行业}`
- `{你的用户}`
- `{你的产品}`
- `{你的服务}`

不要直接把示例品牌、示例场景当成自己的真实资料。
