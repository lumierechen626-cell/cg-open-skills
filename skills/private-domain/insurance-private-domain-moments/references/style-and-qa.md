# 私域朋友圈标题、手机阅读与 QA

## 标题方向

朋友圈标题不要像广告标题。优先使用：

- 真实场景型：今天跟一个客户聊到一半，我停了一下
- 判断型：很多人不是不想配置，是没人把话说清楚
- 反常识型：服务做得越久，越不能急着给答案
- 复盘型：这次交付让我重新改了一个标准

少用：

- “重磅”“必看”“终于来了”
- 夸大结果的标题
- 一眼像课程海报的标题

## 手机阅读规则

- 单段尽量不超过 3 行。
- 每条朋友圈只保留 1 个核心判断。
- 前 3 行要能看出真实场景或明确判断。
- CTA 要轻，不要每条都“欢迎私聊”。
- 标签可放末尾，不能压过正文。

## 发布前 QA

- 这条内容给谁看，是否一眼能看出来。
- 有没有真实发生的事，而不是只有道理。
- 有没有过度暴露客户隐私。
- 有没有把产品、课程或服务吹得太满。
- 有没有一句可以删掉的正确废话。
