# 私域朋友圈内容 Skill

## 定位

这不是“写一条朋友圈”的轻模板，而是一套私域内容工作流。

它默认解决三件事：

- 判断这条朋友圈属于哪一种业务角色。
- 把真实素材整理成能发、能看、能产生关系感的文案。
- 保持风格稳定，不写成品牌宣传、鸡汤小作文或 AI 模板文。

公开版不依赖私有素材库、内部 DNA 文档、固定保存目录或真实客户案例。

## 三种模式

### 模式 A：超级保险人 / B 端交付

适用：

- 学员案例
- 同业反馈
- 交付记录
- 批改反馈
- 课程、咨询、训练后的成果表达

目标：

- 建立交付可信度
- 吸引同业
- 让读者看到你的标准和方法

### 模式 B：客户咨询 / C 端服务

适用：

- 客户咨询过程
- 服务记录
- 保障观念
- 客户真实顾虑
- 方案解释过程

目标：

- 让客户理解你的专业和温度
- 降低客户理解成本
- 让客户觉得“这个人懂我，也能把事讲清楚”

### 模式 C：价值圈 / 长期信任

适用：

- 方法论
- 价值观
- 日常思考
- 行业判断
- 个人工作复盘

目标：

- 建立长期人格资产
- 让关注者形成稳定印象
- 让读者知道你如何判断问题

## 输入要求

至少收这些：

1. 这条朋友圈属于哪种模式。
2. 你想写给谁看。
3. 真实发生了什么。
4. 最想让读者记住哪一句。
5. 希望读者产生什么动作。

如果用户没有明确模式，先判断模式，不要直接开写。

## 品牌标签

默认用占位符，使用者自己替换：

- 模式 A：`#【你的品牌·交付记录】`
- 模式 B：`#【你的品牌·客户咨询】`
- 模式 C：`#【你的品牌·价值圈】`

如果用户不想带标签，输出无标签版。

## 工作顺序

1. 判断模式。
2. 整理素材，不凭空煽情。
3. 明确这条朋友圈想让谁看到什么。
4. 先写标题和开头，再写正文。
5. 最后检查排版、节奏、口气和 CTA。

## 素材整理卡

```markdown
【素材整理卡】
- 当前模式：
- 核心人物：
- 核心事件：
- 最能打动人的细节：
- 最值得记住的一句话：
- 这条的真正目的：
```

如果素材很空，直接提醒用户信息不够，不要硬编戏剧性。

## 三种正文结构

### 模式 A：B 端交付

推荐结构：

1. 标题
2. 学员 / 同业的原始困境
3. 你看到的根因
4. 你做了什么关键动作
5. 对方发生的具体变化
6. 你的交付标准 / 想吸引什么样的人

### 模式 B：客户咨询

推荐结构：

1. 标题
2. 客户处境 / 真实顾虑
3. 为什么客户会困住
4. 你如何帮他把问题讲清楚
5. 结果是什么
6. 给类似人群一句提醒

默认不要写成：

- “恭喜客户成功投保”
- “某某产品真的很好”
- “欢迎私聊了解”

### 模式 C：价值圈

推荐结构：

1. 开头先下判断。
2. 用一个真实场景或反常识切入。
3. 展开 1-3 个理由。
4. 收到一个行动、提醒或价值判断。

## 默认输出

```markdown
【一、模式判断】
- 当前模式：
- 这条的目标：

【二、素材整理卡】
- 核心人物：
- 核心事件：
- 最佳细节：
- 最佳结论：

【三、标题备选】
1. ...
2. ...
3. ...

【四、朋友圈正文】
·

#【你的品牌·标签】

标题

正文

【五、备选开头】
1. ...
2. ...

【六、结尾 / CTA 建议】
- ...

【七、排版检查】
- 是否适合手机阅读：
- 是否有过度营销感：
- 是否还有一句可删：
```

## 禁忌

- 不要硬编客户金句。
- 不要上来就自夸。
- 不要把朋友圈写成新闻稿。
- 不要把产品责任直接当成客户利益。
- 不要只讲自己，不讲读者为什么要看。
- 不要用太重的情绪词。
