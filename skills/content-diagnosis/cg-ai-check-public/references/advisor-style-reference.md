# 顾问内容 AI Check 参考

## 定位

这份 reference 用于保险人、财富顾问、咨询师、私域内容、课程稿、朋友圈、短视频和直播提纲的发布态检查。

它不提供真实客户案例，不包含私有素材库。它只保留公开可复用的口风规则。

## 总原则

专业顾问内容不是把话写得更像专家。

它要做到三件事：

1. 客户听得懂。
2. 客户能代入。
3. 客户看完知道自己为什么要继续听你讲。

最重要的判断：

一段内容如果只有同行觉得专业，客户看完没有感觉，它就不是好内容。

## 应该像什么

像一个真正懂客户处境的顾问，在把复杂问题讲给普通人听。

可以锋利，可以现实，可以刺痛，但不能羞辱人。

可以指出客户的误区，可以指出同行的低级玩法，可以指出行业里的坏机制，但要批行为、批机制、批能力，不骂人格、身份、收入和出身。

## 不应该像什么

不要像：

- 政策课件
- 咨询报告
- 培训讲义
- AI 总结
- 公司宣传稿
- 产品说明书
- 情绪鸡汤
- 纯骂人爽文

## 高风险句式

### 解释腔连接句

高风险：

- 也就是说
- 换句话说
- 本质上
- 从某种意义上说
- 说得更直白一点
- 这背后反映的是
- 这说明了一个问题
- 我们不难发现
- 随着时代发展
- 越来越重要
- 值得注意的是
- 需要强调的是

改法：不要解释前摇，直接给判断。

### “不是……而是……”滥用

这个句式可以用，但不能每隔几段用一次。

高风险：

- 不是 A，而是 B。
- 不是为了 A，而是为了 B。
- 不是简单的 A，而是更深层次的 B。

改法：保留最关键的一两处，其余改成直接判断。

### 讲义腔

高风险：

- 第一，第二，第三。
- 这项能力解决的是……
- 从 A 到 B，再到 C，形成闭环。
- 建立完整体系，实现持续增长。
- 打造个人品牌，实现长期转化。
- 大家一定要理解。
- 我们从以下几个方面展开。
- 这就是完整的方法论。

改法：保留结构，但把“上课”改成“现场”。

## 抽象词替换方向

| 高风险词 | 更好的写法 |
|---|---|
| 内容能力 | 会不会把客户问题讲清楚 |
| 专业价值 | 客户为什么愿意继续听你讲 |
| 信任资产 | 客户一次次觉得你靠谱 |
| 私域经营 | 在客户关系里持续回答问题 |
| 内容闭环 | 这条内容负责筛人，下一条负责解释，最后让客户愿意找你聊 |
| 认知升级 | 客户终于明白自己之前看错了什么 |
| 赋能 | 帮他更快做成某件具体事 |
| 抓手 | 具体动作 / 具体入口 |
| 方法论 | 做法 / 路径 / SOP |
| 客户画像 | 这个客户到底怕什么、要什么、卡在哪里 |

## 顾问内容底线

### 现实伤人，但不侮辱人

允许：

- 指出能力问题。
- 指出低级玩法。
- 指出机制缺陷。
- 指出客户常见误区。

不允许：

- 骂客户蠢。
- 骂同行是废物。
- 嘲笑收入、学历、出身。
- 用人格侮辱替代专业判断。

原则：

- 打能力，不打人格。
- 打行为，不打身份。
- 打机制，不打具体人。

### 不要把客户的脆弱当成交机会

顾问内容经常触碰风险、疾病、养老、负债、现金流、家庭责任。

可以讲风险，但不能利用恐惧。

可以提醒客户不要侥幸，但不能羞辱客户。

可以讲一个家庭没有准备会遇到什么问题，但不能把焦虑夸大成末日。

## 平台口风

### 朋友圈

朋友圈更适合：

- 客户问题回应
- 一线服务观察
- 风险场景翻译
- 个人可信度建立
- 成交后服务证明

不建议：

- 每天贴产品海报
- 天天发收益截图
- 不停晒成交
- 把客户故事写得太像广告

### 私域

私域不是群发广告。它更适合持续教育：

- 为什么要配置？
- 为什么现在做？
- 为什么不能只看收益？
- 为什么不同家庭顺序不一样？
- 为什么买了以后还需要服务？

### 短视频

短视频不是把产品讲短一点，而是把客户痛点讲清楚一点。

一条视频最好只解决一个问题。

开头打中一个真实场景，结尾给一个明确判断。

### 直播

直播不是把人骗进来听产品。

它更适合集中解决疑问，推进认知。

要有主题、有节奏、有答疑、有承接。不要靠喊单、刺激词和恐吓撑场。

## 发布态检查

发布前逐项检查：

1. 开头前 300 字有没有把读者拉进具体问题？
2. 是否一上来就在解释概念、政策或背景？
3. 标题、摘要、导语是否重复说同一句话？
4. 有没有“随着时代发展”“越来越重要”“我们要高度重视”这类 AI 味句子？
5. 有没有太多“不是……而是……”？
6. 有没有内部词，例如“赋能”“抓手”“闭环”“链路”“中台”？
7. 有没有客户听不懂的行业自嗨？
8. 有没有观点很狠，但后文没有证据？
9. 有没有把客户焦虑当成转化工具？
10. 有没有骂人，而不是批行为？
11. 读者看完前三段，能不能知道这篇在讲什么？
12. 新人读完后，能不能复述一句主判断？

## 最终判断标准

一篇稿子清到最后，只问四个问题：

1. 读者看得懂吗？
2. 读者觉得这和自己有关吗？
3. 读者会不会觉得你在利用他？
4. 读者看完会不会更信任你？

如果答案不稳，继续改。
